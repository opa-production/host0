# Fonts Directory

Place your Nunito font files here:

- **Nunito-Regular.ttf**
- **Nunito-SemiBold.ttf**
- **Nunito-Bold.ttf**

You can download Nunito fonts from:
- Google Fonts: https://fonts.google.com/specimen/Nunito
- Or use your own font files

The app requires these three weights:
- Regular (400)
- SemiBold (600)
- Bold (700)

The app will work without the fonts (using system default), but for the best experience, add the font files here.
